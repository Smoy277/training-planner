# Training Planner

**Автор:** [Ислам Токиев]

## Описание

GUI‑приложение для планирования тренировок. Позволяет:
* добавлять тренировки с указанием даты, типа и длительности;
* фильтровать записи по типу тренировки и дате;
* сохранять данные в JSON‑файл;
* загружать данные при запуске приложения.

## Технологии

* Python 3.x
* tkinter (встроенная библиотека для GUI)
* JSON (формат хранения данных)

## Требования

* Python 3.8 или выше
* Стандартная библиотека Python (дополнительные установки не требуются)
