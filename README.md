# 🏋️ Training Planner - План тренировок

**Автор:** Токиев Ислам  
**GitHub:** [https://github.com/Smoy277/training-planner]

## 📖 Описание

Training Planner — это графическое приложение на Python с использованием библиотеки Tkinter для планирования тренировок. Программа позволяет добавлять записи о тренировках, просматривать их, фильтровать по типу и дате, а также сохранять данные в JSON-файл.

## 🚀 Функциональность

- ✅ Добавление тренировок (дата, тип, длительность в минутах)
- ✅ Просмотр всех записей в таблице
- ✅ Фильтрация записей по типу тренировки (выпадающий список)
- ✅ Фильтрация записей по точной дате
- ✅ Удаление выбранных записей
- ✅ Автоматическое сохранение в JSON
- ✅ Загрузка данных при запуске
- ✅ Валидация ввода (формат даты, положительная длительность)

## 🛠 Установка и запуск

### Требования
- Python 3.6 или выше
- Стандартные библиотеки: `tkinter`, `json`, `os`, `datetime`

### Запуск

```bash
# Клонировать репозиторий
git clone [https://github.com/Smoy277/training-planner]

# Перейти в директорию проекта
cd training-planner

# Запустить программу
python training_planner.py
